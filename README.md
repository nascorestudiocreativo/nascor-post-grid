# Nascor Post Grid 🚀

Un plugin para WordPress que genera una cuadrícula de entradas dinámica y moderna[cite: 2]. Cuenta con un buscador en tiempo real y filtros por categoría utilizando AJAX para una experiencia de usuario fluida y sin recargas de página[cite: 2].

## ✨ Características Principales

*   **Búsqueda AJAX:** Buscador de entradas integrado que reacciona automáticamente a partir de los 3 caracteres escritos[cite: 2].
*   **Filtro por Categorías:** Permite a los usuarios filtrar los posts mediante un menú desplegable de categorías de forma dinámica[cite: 2].
*   **Panel de Personalización:** Incluye un panel en el administrador (`Nascor Grid`) para modificar visualmente la cuadrícula[cite: 2]. Permite cambiar:
    *   Colores de fondo superior e inferior (efecto gradiente)[cite: 2].
    *   Color del texto principal[cite: 2].
    *   Colores del botón (fondo y texto)[cite: 2].
    *   Bordes redondeados de las tarjetas en píxeles[cite: 2].
*   **Vista Previa en Vivo:** El panel de configuración incluye un renderizado del shortcode directo en el backend para previsualizar los estilos antes de ir al frontend[cite: 2].
*   **Fácil Integración:** Funciona mediante un simple shortcode para ser ubicado en cualquier página, entrada o widget[cite: 2].

## 📦 Instalación

1. Descarga este repositorio como un archivo `.zip`.
2. En tu panel de WordPress, dirígete a **Plugins > Añadir nuevo > Subir plugin**.
3. Selecciona el archivo `.zip` y haz clic en **Instalar ahora**.
4. Activa el plugin.

## 🛠️ Uso

Copia y pega el siguiente shortcode donde desees mostrar la cuadrícula de entradas:

`[nascor_posts]`[cite: 2]

## ⚙️ Configuración

Dirígete a **Nascor Grid** en el menú lateral de WordPress para ajustar los colores y estilos[cite: 2]. Recuerda que puedes ver los cambios reflejados en la sección de "Vista Previa en Vivo" al guardar los ajustes[cite: 2].
